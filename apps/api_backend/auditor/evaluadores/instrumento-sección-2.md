# **Regla de calificación**

## **Criterio general**
La asignación de 5, 3 o 1 se realiza con base en el número de atributos observables presentes en la FDS para cada ítem.

## **Regla operativa**
- **5**: se observan todos los atributos exigibles del ítem.
- **3**: se observa al menos un atributo exigible del ítem, pero no todos.
- **1**: no se observa ningún atributo exigible del ítem.

## **Regla de aplicación**
- Un atributo se evalúa solo si puede verificarse directamente en la FDS.
- Cuando un atributo depende de una condición previa, se marca como **No corresponde** si esa condición no aparece en la FDS.
- No se permite completar información por inferencia del evaluador.

---

# **Notas de aplicación por ítem**

## **2.0**
- Solo se evalúa el encabezado de la sección.
- No se infiere el título a partir del contenido.

## **2.1**
- Solo se evalúa la clasificación escrita en la FDS.
- No se corrige la clasificación con fuentes externas.
- La información nacional o regional solo se evalúa si aparece en la Sección 2.

## **2.2.1**
- Se acepta pictograma gráfico o nombre del pictograma.
- No se evalúa si el pictograma es correcto frente a la clasificación.

## **2.2.2**
- Solo se evalúa la presencia escrita de la palabra de advertencia.
- No se infiere a partir de pictogramas o frases de peligro.

## **2.2.3**
- Solo se evalúan las indicaciones de peligro escritas en la Sección 2.
- No se infieren a partir de la clasificación.

## **2.2.4**
- Solo se evalúan los consejos de prudencia escritos en la Sección 2.
- No se infieren a partir de recomendaciones generales ubicadas en otras secciones.
- Si el consejo de prudencia existe, pero no cumple la codificación mencionada debe ser evaluado como confiable con restricciones.

## **2.3**
- Expresiones como “ninguno”, “ninguno conocido”, “no se conoce ninguno” o “no relevante” cuentan como presencia del ítem cuando aparecen en la Sección 2.
- No se infieren otros peligros a partir del conocimiento externo del evaluador.
